# Break / Fix Scenarios

## 1. Embedding Dimension Mismatch

Problem:
Vector embeddings generated by the model did not match the FAISS index dimension.

Fix:
Added validation to ensure embedding dimensions match before inserting vectors into the index.

---

## 2. SQLite Schema Migration Failure

Problem:
The metrics database schema changed but existing database files used the old structure.

Fix:
Added initialization logic that creates the table if it does not exist.

---

## 3. Hybrid Score Divide by Zero

Problem:
Hybrid score normalization divided by zero when max score was zero.

Fix:
Added guard condition to prevent division when max score equals zero.
